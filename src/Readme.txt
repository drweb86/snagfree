Building requirements

- Visual Studio 2015.Update 1 Community Edition (with Windows 10 SDK);
- Windows 10 x64;
- https://wixsharp.codeplex.com/ v.Release v1.0.30.0
Unpack its content to folder third-party\WixSharp
(nuget is not working in Republic of Belarus).
- 7-zip x64 installed to default location.

Prior working with SnagFree solution execute 'build third-parties.cmd'.
